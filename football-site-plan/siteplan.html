Football Players Directory - Site Plan

Site Name

Football Players Directory

This name was selected because the website provides a structured directory of professional football players from clubs and countries around the world.

Site Purpose

The site provides detailed information about football players including club, position, nationality, age, and performance statistics. Users can browse, search, and save their favorite players using local storage.

Scenarios

  • Which players are currently playing for a specific club?
  • How can I save and view my favorite football players later?

Color Scheme

  • #1E90FF (Blue) – Headings, buttons, navigation accents
  • #F8F8F8 (Light Gray) – Page background and sections
  • #333333 (Dark Gray) – Body text

Typography

  • Oswald – Used for all headings in this site plan document.
  • Roboto – Used for all body text in this site plan document.
